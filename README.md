# StressForge
**StressForge** - консольное приложение, созданное для упрощения процесса стресс-тестирования программ.
Простой, понятный и быстрый способ проверить корректность и устойчивость алгоритмов.

# Что такое стресс-тестирование
Стресс-тестирование - это метод проверки программы, при котором она запускается на множестве случайно сгенерированных тестов.
Если две версии программы (например, медленная, но надёжная и быстрая, но ошибочная) дают разные результаты - StressForge покажет тест, на котором возникло расхождение.

# Возможности
- Автоматический запуск генератора, корректного и некорректного решений;
- Сравнение результатов работы программ;
- Сохранение теста, на котором найдено различие;
- Не требуется настройки - просто укажите путь к файлам;
- Минимальные требования: не нужны IDE, сторонние программы и подключение к интернету;

# Как это работает
1. Вы указываете три программы:
    - **Корректное решение** - эталонный алгоритм;
    - **Тестируемое решение** - алгоритм, который нужно проверить;
    - **Генератор тестов** - программа, создающая случайные входные данные.
2. StressForge запускает генератор, передаёт его вывод в обоим решениям и сравнивает результаты.
3. Если результаты отличаются - программа показывает тест, на котором найдено несовпадение и сохраняет его в файл.

# Использование
1. Скомпилируйте ваши программы (например `gen.cpp`, `correct.cpp` и `wrong.cpp`) в `.exe` файлы.
2. Запустите StressForge.
3. Укажите путь к каждому из файлов
4. Наблюдайте за процессом - StressForge остановится при нахождении ошибки и покажет тест, на котором программы дали разные ответы.

# Планируемые возможности
- Чтение заранее сохранённых тестов;
- Поддержка пользовательских `checker`-функций для сравнения ответов;
- Встроенная в интерфейс StressForge возможность создания генератора тестов;
- Поддержка запуска `.cpp` и `.py` файлов напрямую.
- Работа с интерактивными задачами;
- Поддержка пользовательских `logger`-ов;
- Многопоточное тестирование;
- Улучшение интерфейса.

# Почему StressForge?
- Простая настройка - без сложных скриптов и IDE;
- Понятный интерфейс и логика работы;
- Подходит для учеников, студентов и олимпиадников;
- Создан программистами для программистов.
